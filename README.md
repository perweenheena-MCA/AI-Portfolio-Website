# AI Website Generation — Task 3

This folder contains a complete, responsive, multi-section website starter demo.

## Files
- `index.html` — Home/About/Services/Impact/Contact sections
- `styles.css` — Styling + responsive layout
- `main.js` — Mobile nav toggle + demo contact form

## How to run (no build tools required)
Open `index.html` directly in a browser, **or** run a simple local server:

```bat
python -m http.server 5500
```
Then open:
- http://localhost:5500
